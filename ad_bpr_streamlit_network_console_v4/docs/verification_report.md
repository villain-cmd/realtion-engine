## 4. 手動変更後の継続性

前回確認済みCPCと次回の媒体設定CSVが異なり、未確認提案とも一致しない場合は `EXTERNAL_MANUAL_CHANGE` として扱う。

- 次回設定CSVの値を新しい正とする
- 適用基準を更新する
- 7日間の観測状態へ入る
- その後の実績CSVを通常ロジックへ戻す

これにより、異常時にRMS/Yahoo管理画面で直接変更した場合でも、次回入力から計算を継続できる。
